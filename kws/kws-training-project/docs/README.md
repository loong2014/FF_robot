This directory collects the operational docs for the KWS training project.

## Docs Index

- [Sample Collection Guide](sample_collection_guide.md)

## Suggested Reading Order

1. Read the sample collection guide first if you are recording data from scratch.
2. Read the data format spec before converting or organizing files.
3. Read the training checklist right before running `prepare_data.py` and `train_kws.py`.
